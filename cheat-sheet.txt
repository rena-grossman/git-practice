ECHO = is for repeating things.
CAT = prints content of a file to the screen.
CD = change directory.
PWD = print working directory.
LS = lists files and folders.
CD .. = go back (up a level).
CD~ = go to home directory.
TOUCH= see or create file.
MKDIR = new folder.
MV = move. example:  cheat-sheet.txt projects
GREP = way to search data. Example: |grep -i 'paris' will give every entry with that word. Example 2: |grep -1 'paris' | wc -1 is number of lines.
